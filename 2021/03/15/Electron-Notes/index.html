Electron 简介

Electron 基于 Chromium 和 Node.js, 让你可以使用 HTML, CSS 和 JavaScript 构建应用。

进程和线程

什么是 Process - 进程

一个正在被执行的程序对象就是 一个进程。可以看作是 一个车间。

什么是 Thread - 线程

线程 是操作系统能够进行运算调度的最小单位。它被包含在进程之中，是进程中的实际运作单位。可以看作是 车间里的工人。

进程和线程之间的区别

  • 内存使用方面的区别
  • 通信机制方面的区别
  • 量级方面的区别

Electron 中的主进程和渲染进程

主进程：

整个 Chromium 进程就是主进程。

  • 主进程只有一个，是整个应用程序的指挥官，作为整个程序的入口
  • 可以使用和系统对接的 Electron API ，如：创建菜单、上传文件等
  • 全面支持 Node.js
  • 可以创建渲染进程

渲染进程：

Chromium 中的 tab 就是渲染进程。

  • 渲染进程可以有多个，每个对应一个窗口
  • 每个都是单独的进程
  • 全面支持 Node.js 和 DOM API
  • 可以使用一部分 Electron API
