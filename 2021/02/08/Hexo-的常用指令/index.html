二、配置主题

主题可以自己去 github 上找，下面以 hexo-theme-yilia 为例。
先将其 clone 到 myBlog 的 themes 文件夹下面。然后修改 _config.yml 配置文件，找到 theme，修改为对应的文件夹名称即可！其余配置可以参考对应主题的 readme 使用。

关于修改主题的配置文件：其内部默认的相对根目录为 public
