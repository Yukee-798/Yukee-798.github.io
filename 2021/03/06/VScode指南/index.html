快捷键记录

快捷键 说明
option + command + Z 返回撤销
option + shift + A 注释选中代码
command + shift + K 删除行
command + shift + ➡️ 选取行
command + D 选取最近的相同变量
command + shift + L 选取所有与当前选中内容相同的部分
command + U 撤销上次光标选取
command + G 找查下一个
command + shift + G 找查上一个
command + W 关闭当前文件
